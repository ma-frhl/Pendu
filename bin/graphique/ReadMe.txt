Programmation en mode graphique

La création d'affichages graphiques est réalisable
au moyen de l'API EcranGraphique développée à cette fin.
Elle est stockée dans l'archive EcranGraphique.Class.zip
et est composée de trois fichiers .class :
 - EcranGraphique.class
 - EcranGraphique$MyFrame.class
 - EcranGraphique$MyCanvas.class
Ces trois fichiers doivent être extraits dans le répertoire
où sera placé le fichier .java qui doit utiliser l'API.

Deux exemples d'utilisation sont donnés dans l'archive
EcranGraphique.Demos.zip.

Une documentation au format html/javadoc est contenue
dans l'archive EcranGraphique.Docs.zip
dont il convient de réaliser l'extraction.
Cette opération a pour conséquence la création
d'un répertoire nommé Docs contenant
une quinzaine de fichiers.
Entre autres fichiers, ce répertoire contient le fichier
index.html qui peut être ouvert dans n'importe quel
navigateur internet et qui est la racine de la documentation.
